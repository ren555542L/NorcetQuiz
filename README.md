# NorQuiz — NORCET Exam Quiz Platform

NorQuiz is a web-based quiz platform designed for students preparing for the AIIMS NORCET examination.

The platform provides previous-year NORCET questions in an interactive quiz format, helping users practise exam-style questions, track their progress, and review their mistakes.

Users can choose the number of questions, filter questions by year, shuffle the quiz, mark questions for review, and navigate between questions using a question palette.

After completing the quiz, the platform displays the final score, number of correct, incorrect, and unanswered questions, along with the total time taken. Users can also review every attempted question and compare their selected answer with the correct answer.

100-question quiz modes
Year-wise question filtering
Random question shuffling

Previous and Next navigation
Question palette for quick access

Quiz progress bar

Correct, incorrect, and unanswered question summary

Purpose

NorQuiz aims to make NORCET preparation more organised and interactive by converting previous-year question papers into a simple digital quiz experience. It helps nursing aspirants practise regularly, identify weak areas, and become familiar with the exam pattern.
